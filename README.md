# StayBoost Agent

Interni MVP aplikace pro spravu leadu malych hotelu, penzionu a apartmanu. Prvni verze je ciste frontendova React aplikace bez backendu, loginu a automatickeho rozesilani e-mailu.

## Spusteni

```bash
npm install
npm run dev
```

## Validace

```bash
npm run build
npm run lint
npm run typecheck
```
